# Palindrome Algorithm — 4 Prompts, 4 Developer Levels

Same problem, different prompts. The quality of the prompt determines the quality of the result.

---

## 1. Junior Developer

```
Write a function that checks if a word is a palindrome.
```

---

## 2. Intermediate Developer

```
Write a function isPalindrome that takes a string and returns true if it reads
the same forwards and backwards, false otherwise. Choose the most appropriate
algorithm for this problem. Handle edge cases like empty strings and single
characters.
```

---

## 3. Expert Developer

```
Implement isPalindrome(_ s: String) -> Bool in Swift.

Requirements:
- O(n) time, O(1) extra space — no reversed copy
- Use Unicode scalars for correctness with emoji and accented characters
- Handle edge cases: empty string, single char, even/odd length
- Write unit tests
- Do not create any files — output code in the conversation only
```

---

## 4. AI Developer (/algorithm)

```
/algorithm Write a function that checks if a word is a palindrome.
```
